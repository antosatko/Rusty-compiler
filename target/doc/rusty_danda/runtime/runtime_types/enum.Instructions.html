Variants

Debug(usize)

debug reg | prints value of reg()

Wr(usize)

write stack_offset | moves value from reg(0) to stack(stack_end - <stack_offset>)

Rd(usize, usize)

read stack_offset reg | reads value from stack(stack_end - <stack_offset>) to its reg()

Wrp(usize, usize)

write pointer | moves value from reg() to stack()

Rdp(usize, usize)

read_pointer pointer_reg reg | reads value from stack(reg1) to its reg()

Rdc(usize, usize)

read constant | reads value from stack(<stack_pos>) to its reg()

Ptr(usize)

pointer stack_pos | stores pointer to stack(stack_end - <stack_offset>) in reg(0)

Idx(usize, usize)

Index pointer idx | gets pointer from reg() repairs it and adds reg()

Repp(usize)

Repair pointer | Repairs pointer in reg()

Alc(usize, usize)

allocate pointer size_reg | reserves on heap and stores location in registers()

Dalc(usize)

deallocate pointer | frees heap()

RAlc(usize, usize)

reallocate pointer size_reg | resizes heap() for ; additional space is filled with null

Goto(usize)

goto pos | moves code_pointer to

Gotop(usize)

goto pos_reg | moves code pointer to reg()

Brnc(usize, usize)

branch pos1 pos2 | if reg(0), goto else goto

Ret

return | moves code_pointer to the last position in stack retrieved from stack

Res(usize)

reserve size | reserves on stack and saves current reg(0)

Mov(usize, usize)

Add

add | reg(0) is set to the result of operation: reg(0) + reg(1)

Sub

subtract | reg(0) is set to the result of operation: reg(0) - reg(1)

Mul

Div

divide | reg(0) is set to the result of operation: reg(0) / reg(1)

Mod

modulus | reg(0) is set to the result of operation: reg(0) % reg(1)

Equ

equals | reg(0) is set to the result of operation: reg(0) = reg(1)

Grt

greater than | reg(0) is set to the result of operation: reg(0) > reg(1)

Less

less than | reg(0) is set to the result of operation: reg(0) < reg(1)

And

and | reg(0) is set to the result of operation: reg(0) & reg(1)

Or

or | reg(0) is set to the result of operation: reg(0) | reg(1)

Not

not | reg(0) is set to the result of operation: !reg(0)
